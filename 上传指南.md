# 🦞 历史纪念网站手动上传指南

## 问题
GitHub 连接超时，自动推送失败。

## 解决方案：手动上传

### 方法 1: GitHub 网页上传（推荐）

1. **打开 GitHub**
   - 访问：https://github.com/dztgfyxgs-glitch/history-memorial

2. **上传文件**
   - 点击「Add file」→「Upload files」
   - 把 `index.html` 拖进去
   - 点击「Commit changes」

3. **等待生效**
   - 等 1-2 分钟
   - 访问：https://dztgfyxgs-glitch.github.io/history-memorial/

---

### 方法 2: 使用 GitHub Desktop

1. **下载 GitHub Desktop**
   - https://desktop.github.com/

2. **克隆仓库**
   - File → Clone repository
   - 选择 dztgfyxgs-glitch/history-memorial

3. **复制文件**
   - 把 index.html 复制到克隆的文件夹

4. **提交推送**
   - 写提交信息
   - 点击 Commit
   - 点击 Push

---

### 方法 3: 稍后重试命令行

等网络好转后，运行：

```bash
cd ~/Desktop/history-memorial
git push -u origin main
```
